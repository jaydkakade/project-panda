# 🐼 Project Panda

> An interactive birthday story website crafted with ❤️ using HTML, CSS, and JavaScript.

## 🌙 About

Project Panda is a magical one-page interactive birthday experience that takes the user through a heartwarming adventure with a cute cat 🐱 and panda 🐼.

Instead of simply saying "Happy Birthday", the website tells a story through animations, playful interactions, and an emotional ending.

---

## ✨ Features

- 🌌 Animated night sky
- ⭐ Twinkling stars
- 🌙 Glowing moon
- 🐱 Cute cat companion
- 🐼 Friendly panda guide
- ⌨️ Typewriter animations
- 🎮 Interactive friendship quiz
- 🎁 Surprise mini-game
- 🌠 Wish upon a star
- 💌 Emotional birthday letter
- 🎂 Confetti celebration
- 📱 Mobile responsive

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
